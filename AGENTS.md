# Agent Instructions

This repo is an AI-assisted resume tailoring workflow.

Before doing any workflow task, read `CHAT_BOOTSTRAP.md` and follow its stage routing.

Core rules:
- Keep real resumes, evidence, role briefs, generated PDFs, recruiter notes, and workflow state out of Git.
- Use `sample/` files only for demos.
- For real runs, use the ignored mirrored folders: `input/`, `evidence/`, `workflow_state/`, `Gmail/role_briefs/`, and `output/`.
- Do not invent experience, employers, tools, metrics, ownership, credentials, or outcomes.
- End every workflow stage with `Next options:`.
