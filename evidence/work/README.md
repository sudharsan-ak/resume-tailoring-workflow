# Work Evidence

Use this folder for role/workplace evidence from your main professional experience.

The public template uses the neutral folder name `work`. If your real local workflow uses employer-specific subfolders, keep those populated files ignored and do not commit employer-specific names unless they are intentionally public.

## Recommended Files

```text
INDEX.md
EVIDENCE_ROUTER.md
EVIDENCE_CHEATSHEET.md
evidence_manifest.json
E01_<short_slug>.md
E02_<short_slug>.md
```

## Suggested Use

- `INDEX.md`: fast list of all evidence IDs and titles.
- `EVIDENCE_ROUTER.md`: maps job themes to evidence IDs. See `EVIDENCE_ROUTER_TEMPLATE.md` for a worked example.
- `EVIDENCE_CHEATSHEET.md`: human-readable summary of the strongest evidence.
- `evidence_manifest.json`: structured routing metadata.
- `E##_<slug>.md`: one detailed evidence record per achievement/system/project.

Only template files are committed here. Real `E##` files are ignored.
