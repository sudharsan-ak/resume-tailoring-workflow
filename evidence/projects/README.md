# Project Evidence

Use this folder for personal projects, portfolio projects, open-source work, or side projects that may appear on the resume.

Real project files are ignored by Git. Only templates and guide files are committed.

Recommended local structure:

```text
evidence/projects/
  INDEX.md
  P01_task_queue_demo.md
  P02_metrics_viewer_demo.md
```

## How Project Evidence Is Used

Project evidence helps decide:

- which projects belong on a tailored resume
- which project should appear first
- which bullets should be rewritten for a JD
- when a project should be removed to protect one-page layout

Projects should be treated like evidence, not filler.
