# Tailored Count Template

Copy this to `workflow_state/tailored_count.md` for real local use.

Do not commit the populated real file.

Purpose:
- Track how many brand-new role-specific resumes were built on each date.
- Do not increment this count for rebuilds or minor edits to an already-tailored role.

Format:

```text
MM/DD/YYYY

Count - N
```

Example:

```text
05/14/2026

Count - 3
```
